# react-native-demo
react native tech demo

## Install

```
git clone https://github.com/rg-external/react-native-demo.git
cd react-native-demo
npm install
```

## Test run

#### ios
```
run-ios
```

#### android
```
run-android
```

## build

#### build
```
cd android && ./gradlew assembleRelease
```

#### build and install device
```
cd android && ./gradlew installRelease
```
